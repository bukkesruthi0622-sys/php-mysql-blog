# PHP MySQL Blog Application

This project is a basic blog application developed using PHP and MySQL.

## Features
- User Registration and Login
- Password hashing for security
- Session management
- Create, Read, Update, Delete (CRUD) operations
- MySQL database integration

## Tools Used
- PHP
- MySQL
- XAMPP
- GitHub

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder inside htdocs
4. Import database and tables
5. Run http://localhost/blog/index.php
